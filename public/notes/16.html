Getting your OpenAI API key

To get a OpenAI API key, follow these instructions:

1. Login to openai.com.

2. Click on "Personal" on the top right hand corner.

3. Click "View API keys"

4. Click "Create new secret key and copy it"

Warning: OpenAI API keys are not free past the $15 free trial, See openai.com/pricing for more infomation
